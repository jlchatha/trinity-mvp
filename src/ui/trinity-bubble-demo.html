API Documentation

Quick reference for using the Trinity Bubble component.

// Create a basic bubble const bubble = new TrinityBubble({ target: document.getElementById('my-button'), content: 'Hello Trinity!', position: 'top', theme: 'default', trigger: 'hover' }); // Static helper methods TrinityBubble.tooltip(element, 'Tooltip text'); TrinityBubble.notification('Success message', { theme: 'success' }); // Update content dynamically bubble.updateContent('New content'); bubble.show(); // Manual show bubble.hide(); // Manual hide bubble.destroy(); // Clean up

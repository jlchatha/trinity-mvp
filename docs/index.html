Frequently Asked Questions

Everything you need to know about Trinity MVP

What makes Trinity different from other AI assistants?

Trinity MVP combines four key advantages:

  1. Persistent Memory: Trinity remembers conversations and context across sessions, eliminating the need to reintroduce yourself or your projects.
  2. Local File Integration: Direct access to your file system allows Trinity to read, write, and modify files as part of natural conversations.
  3. Hybrid Local-First Privacy: Conversation history stays local, while smart context selection ensures efficient API requests.
  4. Transparent Design: Trinity shows you exactly what it remembers and how it optimizes your context, with no black box processing.

How does Trinity's memory system work?

Trinity uses a sophisticated 4-tier memory system:

  • Core Memory: Fundamental information about you and your preferences
  • Working Memory: Recent conversations and active projects
  • Reference Memory: Important project details and key information
  • Historical Memory: Archived conversations and completed projects

The system automatically manages these tiers, moving information between them based on relevance, recency, and importance. You can view and manage this memory through Trinity's Memory Explorer panel.

What are Trinity's privacy and data policies?

Trinity follows a hybrid local-first approach:

  • Your conversation history is stored locally on your machine
  • Current requests and selected context are sent to the Claude API
  • Trinity optimizes what context is included in API requests
  • No data is collected or used for model training
  • You can see exactly what Trinity remembers through the Memory Explorer
  • Trinity is transparent about data flow and does not make misleading privacy claims

What platforms does Trinity support?

Trinity MVP currently supports Linux and macOS. A Windows version is planned for future releases. Trinity runs as a native desktop application for optimal performance and system integration.

What does Trinity require to run?

To use Trinity MVP, you'll need:

  • A Linux or macOS system
  • Claude Code CLI installed
  • An Anthropic API key
  • Internet connection for API requests
  • Approximately 100MB of disk space

The setup process takes about 5 minutes and is automated through our installation scripts.
